Manakrishi Case Study

Manakrishi – Farmer Support Mobile App

A precision drone spraying platform and app that helps farmers access crop guidance, reducing manual spraying health risks and pesticide waste.

Tech Stack

  • Frontend: React Native
  • Backend: Node.js, Express
  • Database: MongoDB
  • Hardware: Drone IoT Integration

Key Features

  • Precision drone spraying scheduling
  • Real-time crop health monitoring
  • Multi-language support for rural farmers
  • Secure user authentication

System Architecture

React Native application sends REST API requests to a Node.js backend hosted on AWS. Real-time drone telemetry is processed via WebSockets and stored in MongoDB for historical yield analysis.

Impact / Metrics

  • 100% health safety ensured for field workers
  • Optimized crop yields by reducing pesticide waste
  • Currently serving active test users in target regions

Challenges & Solutions

Challenge: Integrating real-time drone telemetry with low-latency updates for farmers.
Solution: Transitioned from standard HTTP polling to WebSocket connections for immediate status delivery.
